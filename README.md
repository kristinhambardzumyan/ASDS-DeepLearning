# ASDS-DeepLearning
Final project for the Deep Learning course within the Applied Statistics and Data Science (ASDS) program at Yerevan State University. The project focuses on duplicate question detection using the Quora Question Pairs dataset.

## Project Topic
Duplicate Question Detection using the Quora Question Pairs (QQP) dataset.

## Files
- `QQP_Duplicate_Question_Detection.ipynb` – Full implementation
- `report.pdf` – Final project report

## Notes
Models were trained using Google Colab with GPU support.
